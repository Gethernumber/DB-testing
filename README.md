# Тестирование баз данных: SQL & NoSQL

В этом репозитории собраны примеры моих запросов и тестовых артефактов, выполненных в рамках работы с базой данных учебного веб-приложения **DemoShopping**.

### Стек инструментов:

- **SQL:** DBeaver, MySQL Workbench.
- **NoSQL:** MongoDB.

### Содержание:

- **[SQL запросы (DBeaver):](SQL.md)**
  - Практика написания выборок любой сложности (`SELECT`);
  - Работа с объединением таблиц (`INNER JOIN`, `LEFT/RIGHT JOIN`) для проверки связей между данными;
  - Фильтрация, сортировка и группировка данных.
- **[NoSQL запросы (MongoDB):](NoSQL.md)**
  - Базовые операции CRUD (создание, чтение, обновление, удаление документов);
  - Практика написания команд в консоли MongoDB.
